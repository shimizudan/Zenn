# サイコロで円周率を推定する - シミュレーションコード

このディレクトリには、浜松医科大2025年入試問題を題材にした、サイコロによる円周率推定のシミュレーションコードが含まれています。

## ファイル構成

- `dice_pi_simulation.jl`: メインのシミュレーションコード
- `grid_visualization.png`: 格子点による円の近似の可視化
- `error_heatmap.png`: 試行回数と分割数による誤差のヒートマップ
- `convergence_comparison.png`: 問題(3)と問題(4)の方法の収束比較

## 実行方法

```bash
julia dice_pi_simulation.jl
```

## 必要なパッケージ

```julia
using Pkg
Pkg.add(["Random", "StatsBase", "Plots", "Printf"])
```

## 実行結果の概要

### Part 1: サイコロによる均等分布生成
- 1〜5の均等生成（6が出たらやり直し）
- 1〜7の均等生成（ゾロ目を7とする）

### Part 2: モンテカルロ法による円周率推定
- 問題(3)の方法：試行回数のみ増加
- 問題(4)の方法：分割を細かくする
- 総合シミュレーション：両方の効果を比較

### Part 3: 可視化
- 格子点による円の近似
- 誤差のヒートマップ
- 収束比較グラフ

## 重要な知見

**問題(3)の方法（試行回数のみ増加）では精度が根本的に向上しない**

理由：同じ格子点（100×100）を繰り返しサンプリングするだけで、空間の分解能が固定されているため。

**問題(4)の方法（分割を細かくする）が効果的**

理由：サンプリング空間の分解能が向上し、真の円周率に近づく。
