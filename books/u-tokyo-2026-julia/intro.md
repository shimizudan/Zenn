## この本について

2026年の東京大学理科（理系）の数学の入試問題を，Julia言語を用いて考えたり確認してみました。

数学の入試問題というのは，当然頭の中で考えて計算で解く必要があります。コンピューターを使って解くわけにはいきません。ですが，こういったツールを用いて考えたときにどのように考えるのか，また答えについてもどのように確認するのかというのは，自分の知見を与えてくれます。

私自身が受験生を指導していた立場として，2026年度の東京大学の問題は非常に難しかったと思います。私が関わってきた中でも本当に難しかった年度に入るのかな，と考えています。ですが，難しいと言っても，その中にも面白い題材がたくさん含まれています。Julia言語を使うとどのように考えられるか，また計算結果をどのように確認できるか，そういったことをまとめたいと思っています。

## Julia言語について

[Julia](https://julialang.org/) は，数値計算・科学技術計算に特化したプログラミング言語です。Pythonのような書きやすさと，C言語のような実行速度を兼ね備えています。数学的な記述に近いコードが書けるため，数学の問題を考えるツールとして非常に適しています。

本書では，主に以下のパッケージを利用しています：

| パッケージ | 用途 |
|---|---|
| `CairoMakie.jl` | グラフ・図の描画 |
| `LaTeXStrings.jl` | 図中への数式表示 |
| `Optim.jl` | 最適化（最大値・最小値の探索） |
| `QuadGK.jl` | 数値積分 |
| `Combinatorics.jl` | 組み合わせ計算 |
| `ForwardDiff.jl` | 自動微分（導関数の計算） |
| `Roots.jl` | 方程式の数値解法 |
